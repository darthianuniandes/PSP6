Ejecute el programa desde el directorio raíz(aquí se encuentran la carpeta src y target) utilizando el comando: 

mvn package
java -cp target/PSP7RegSimp-1.0-SNAPSHOT.jar co.com.ism.psp7.controller.MainController


La aplicación soporta que ingrese la ruta completa de un archivo llamado “numeros.txt”. Si no provee la ruta utilice el archivo provisto en la raíz del proyecto entregado. El programa leerá este archivo por defecto.

La documentación se encuentra en el directorio target/site
